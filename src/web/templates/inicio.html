Que ofrece

  • Busqueda full-text en espanol con filtros por CPV, NUTS, tipo de contrato, estado, importe y mas
  • Detalle completo de cada licitacion: criterios de adjudicacion, requisitos de solvencia, lotes y documentos
  • Perfiles de organos y empresas con estadisticas de contratacion, importes y CPV frecuentes
  • Inteligencia competitiva: licitaciones similares, estadisticas de precios y adjudicatarios frecuentes
  • Catalogos para descubrir valores validos de cada filtro disponible en la API

Por que licit

La PLACSP publica datos de contratacion publica a traves de feeds Atom sin capacidad de filtrado. Las alternativas existentes forman parte de planes enterprise de plataformas privadas. licit estructura estos datos en una API abierta, pensada para desarrolladores, analistas y agentes.
